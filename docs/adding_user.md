### Overview
BirdDog is built for collaboration. Whether you are onboarding a new AE, an SDR, or a Sales Manager, the **System Setup** page allows you to control exactly what each user can see and do within the platform. Adding a user triggers an automated welcome email with their credentials and onboarding documentation.

### How to Add a User (0:12 â€“ 2:15)

1.  Navigate to **System Setup** in the bottom-left sidebar.
2.  Click the **Add a User** button.
3.  **Choose the Role:** Select from Normal, View-Only, or Manager.
4.  **Enter Email:** Input the user's professional email address.
5.  **Configure Visibility:** * If adding a **Normal** user, choose if they should be able to view other teammates' accounts.
    * If adding a **View-Only** user, select the specific teammate they are supporting.
6.  **Admin Toggle:** For Normal users, choose "Yes" or "No" for Admin access. Admins can access:
    * **Assignment Desk** (Farsight recommendations)
    * **Assignment Progress** (Team performance)
    * **System Setup** (Integrations and Signals)
7.  **Submit:** Click **Create**.

---

### Onboarding Experience
Once you click **Create**, BirdDog automatically handles the rest:
* **Welcome Email:** The new user receives an email with their login credentials.
* **Support Resources:** The email includes help documentation to get them up to speed quickly.
* **Immediate Access:** They can log in and begin researching accounts right away.
